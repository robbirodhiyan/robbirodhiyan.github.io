# Ahmad Robbi — Updated Portfolio Website

A modern, responsive, ATS-aligned personal portfolio for:

- Enterprise Application Lead
- Technical Lead
- Senior Full-Stack Developer
- SAP / Business Systems Integration roles

## Included

- Responsive one-page portfolio
- Dark/light theme
- Enterprise case studies and project filters
- Updated career impact, experience, and technology stack
- Downloadable CV and PDF portfolio
- SEO metadata and JSON-LD profile schema
- GitHub Pages compatible static files

## Deploy to GitHub Pages

1. Back up the current repository.
2. Replace the repository files with the contents of this folder.
3. Commit and push to the branch used by GitHub Pages, commonly `master` or `main`.
4. Open `https://robbirodhiyan.github.io/` and hard-refresh the browser.

No build process is required.
