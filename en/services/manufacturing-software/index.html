FAQ

Frequently asked questions.

Final answers may vary based on scope, system access, and project requirements.

Can OEE and Yield formulas be customized?

Yes. Formulas and business rules are validated with users because each factory may use different definitions and data sources.

Can the dashboard use SAP data?

Yes. Data can be retrieved through APIs or an integration layer that follows company policies.

Can it be used on tablets or mobile devices?

Yes. The interface is responsive and can be adapted for operators, supervisors, and management.

Can the project start with one production line?

Yes. It can begin as a pilot for one line or process and expand incrementally.
